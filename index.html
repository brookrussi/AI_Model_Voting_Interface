üöÄ Quick Setup Guide

Set up Supabase Database

Run the SQL schema in your Supabase project to create the required tables.

psql -h your-project.supabase.co -U postgres -d postgres -f supabase_schema.sql

Update Supabase Keys

Replace the placeholder API keys in the HTML files with your actual Supabase credentials.

const SUPABASE_ANON_KEY = 'your_actual_anon_key_here';

Import Your Conversations

Use the Python parser to import your OpenRouter markdown exports.

python parse_openrouter_md.py "path/to/your/exports/" --supabase-key YOUR_KEY

Deploy to GitHub Pages

Push all files to your GitHub repository and enable GitHub Pages in settings.

git add .
git commit -m "Deploy blind voting system"
git push origin main
